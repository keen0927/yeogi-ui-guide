UI가이드 소개

여기어때 UI가이드를 소개합니다.
본 컨텐츠는 부트스트랩4를 기반으로 제작 되었으며 사용방법 또한 부트스트랩4와 동일합니다.
추가적으로 자주사용하는 자바스크립트 라이브러리와 여기어때 서비스에서 자주 사용되는 모듈들의 사용법과 테스트가능한 샘플코드를 제공합니다.
다운로드 패키지는 기본 backoffice 템플릿이며 UI가이드를 통하여 프론트화면을 개발할수있도록 제작되었습니다.

기본샘플 다운로드

로컬 개발 설정

샘플파일 다운로드 후 사이드네비게이션(snbControl.js)의 경로를 수정해야 정상 렌더링된 화면을 확인할 수 있습니다.
static디렉토리의 정적요소들의 경로를 서버환경에 맞게 수정이 필요합니다. (기본 설정은 로컬환경의 상대경로로 설정되어 있음.)
추가 적으로 기본 사이드네비게이션작업시 snbControl.js로만 메뉴 구성이 가능하며
직접 개발이 필요할 경우 해당 페이지를 참고 해 주세요. 링크

Live-server 설정
VSCODE로 로컬작업시 live-server를 이용하여 저장시 자동으로 화면이 갱신이 되는 형식으로 화면개발을 할 수 있습니다.
  1. 1. live-server 전역 설치
  2. 2. VSCODE EXTENSION에서 live-server를 검색하여 설치후 리스타트.
  3. 3. VSCODE 하단 우측의 스테이터스 바에서 Go Live를 클릭
// live-server 전역 설치
npm i -g live-server
초기 페이드인 효과 설정 끄기
기본 설정은 초기 진입시 페이드인 효과 & 스피너를 사용한다.
이 효과를 끄려면 .wrap 태그에 data-fade="off" 를 추가 하면 된다. 
<div class="wrap" data-fade="off">
