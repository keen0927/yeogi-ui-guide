UI가이드 소개

여기어때 UI가이드를 소개합니다.
본 컨텐츠는 부트스트랩4를 기반으로 제작 되었으며 사용방법 또한 부트스트랩4와 동일합니다.
추가적으로 자주사용하는 자바스크립트 라이브러리와 여기어때 서비스에서 자주 사용되는 모듈들의 사용법과 테스트가능한 샘플코드를 제공합니다.
다운로드 패키지는 기본 backoffice 템플릿이며 UI가이드를 통하여 프론트화면을 개발할수있도록 제작되었습니다.

기본샘플 다운로드 후 설정

처음 설정해주어야 하는부분은 font디렉토리의 자원을 임의의 사용자가 작업하는 작업경로로 이동 하고,
CSS의 아이콘팩과 잘난체 경로를 절대경로 수정 해 주세요.

1. static > fonts 의 파일들을 사용자가 작업하는 작업경로의 자원 디렉토리로 옮긴후,
2. static > css > service 디렉토리의 font-icon-load.css 안의 절대경로를 1에서 이동한 경로로 수정 해 주세요.

마지막으로 index.html최하단의 JS의 snbControl.js 경로 또한 변경 해주세요.

다운로드 버전은 메뉴가 자바스크립트 렌더링으로 작업이 되어있습니다.
타임리프나 다른 템플릿을 사용하고 싶으시면 HTML버전을 참고해서 작업 해주세요. 

/* 아래의 상대경로를 사용자가 작업하는 환경의 절대경로로 수정 해 주세요. */

/* 잘난서체 */
@font-face {
	font-family: 'yg-jalnan';
	src:  url('../../fonts/yg-jalnan.eot');
	src:  url('../../fonts/yg-jalnan.eot?#iefix') format('embedded-opentype'),
		url('../../fonts/yg-jalnan.ttf') format('truetype'),
		url('../../fonts/yg-jalnan.woff') format('woff');
	font-weight: normal;
	font-style: normal;
}

/* 아이콘팩 */
@font-face {
    font-family: 'icomoon';
    src: url('../../fonts/icomoon.eot');
    src: url('../../fonts/icomoon.eot#iefix') format('embedded-opentype'), url('../../fonts/icomoon.ttf') format('truetype'), url('../../fonts/icomoon.woff') format('woff'), url('../../fonts/icomoon.svg#icomoon') format('svg');
    font-weight: normal;
    font-style: normal;
}
