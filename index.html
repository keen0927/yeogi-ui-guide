UI가이드 소개

여기어때 UI가이드를 소개합니다.
본 컨텐츠는 부트스트랩4를 기반으로 제작 되었으며 사용방법 또한 부트스트랩4와 동일합니다.
추가적으로 자주사용하는 자바스크립트 라이브러리와 여기어때 서비스에서 자주 사용되는 모듈들의 사용법과 테스트가능한 샘플코드를 제공합니다.
다운로드 패키지는 기본 backoffice 템플릿이며 UI가이드를 통하여 프론트화면을 개발할수있도록 제작되었습니다.
CSS

CSS를 로드하기 위해 아래의 코드를 <head>의 타 CSS보다 상위에 위치하게 붙여넣기 하세요.

<link rel="stylesheet" href="https://static.goodchoice.kr/static/ui/css/library/bootstrap.min.css">
<link rel="stylesheet" href="https://static.goodchoice.kr/static/ui/css/service/style.css">
JS

플러그인들은 Javascript를 사용해야 작동합니다. 기본적으로 jQuery, Popper.js 및 자체 자바스크립트 플러그인이 필요합니다.
아래의 script를 <body>태그 바로 앞에 붙여 넣기 해주세요. 순서는 아래와 같이 배치 해야 정상 작동 합니다.

// 라이브러리
<script src="https://static.goodchoice.kr/static/ui/js/library/jquery-3.3.1.min.js"></script>
<script src="https://static.goodchoice.kr/static/ui/js/library/popper.min.js"></script>
<script src="https://static.goodchoice.kr/static/ui/js/library/bootstrap.min.js"></script>
<script src="https://static.goodchoice.kr/static/ui/js/library/iscroll.js"></script>
<script src="https://static.goodchoice.kr/static/ui/js/library/datatables.js"></script>

// 서비스
<script src="https://static.goodchoice.kr/static/ui/js/service/common.js"></script>
기본 템플릿

다음은 HTML5 기본 템플릿을 적용한 샘플 페이지 입니다. 

<!DOCTYPE html>
<html lang="ko">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>여기어때 - 표준 템플릿 가이드</title>
    <link rel="stylesheet" href="https://static.goodchoice.kr/static/ui/css/library/bootstrap.min.css">
    <link rel="stylesheet" href="https://static.goodchoice.kr/static/ui/css/service/style.css">
</head>

<body>

    <!-- Wrap -->
    <div class="wrap">

        <!-- Container -->
        <div class="container">

            Hello, World!

        </div>
        <!-- //Container -->

    </div>
    <!-- //Wrap -->

    <!-- JS : library  -->
    <script src="https://static.goodchoice.kr/static/ui/js/library/jquery-3.3.1.min.js"></script>
    <script src="https://static.goodchoice.kr/static/ui/js/library/popper.min.js"></script>
    <script src="https://static.goodchoice.kr/static/ui/js/library/bootstrap.min.js"></script>
</body>

</html>
