부트스트랩 시작

부트스트랩 사용시 알아두어야 할 기본 CSS클래스들에 대한 설명 입니다.
  1. 1. 간격(마진, 패딩) 조정
  2. 2. 그리드 설명
  3. 3. 텍스트 컨트롤
간격(마진, 패딩) 조정
엘리먼트의 마진, 패딩값 조절은 사용방법이 같다.
마진은 m-수치
패딩은 p-수치 수치는 1에서 5까지 조절 할 수 있다.

마진,패딩의 상,하,좌,우 값에만 주고 싶을때는 p방향-수치, m방향-수치를 사용한다.
방향은 top, right, bottom, left의 앞글자인 t,r,b,l을 사용한다. 
// 마진값 조정 접두사 m은 margin의 m 이며 뒤의 수치값으로 간격 조절이 가능
<div class="m-0">마진값 조정</div>
<div class="m-1">마진값 조정</div>
<div class="m-2">마진값 조정</div>
<div class="m-3">마진값 조정</div>
<div class="m-4">마진값 조정</div>
<div class="m-5">마진값 조정</div>

// 패딩값 조정 접두사 p는 padding의 p 이며 뒤의 수치값으로 간격 조절이 가능
<div class="p-0">패딩값 조정</div>

// 상,하,좌,우 특정 부분에만 값을 지정하고 싶을때
<div class="mt-1">마진값 상단 조정</div>
<div class="pt-1">패딩값 상단 조정</div>

<div class="mr-1">마진값 우측 조정</div>
<div class="pr-1">패딩값 우측 조정</div>

<div class="mb-1">마진값 하단 조정</div>
<div class="pb-1">패딩값 하단 조정</div>

<div class="ml-1">마진값 좌측 조정</div>
<div class="pl-1">패딩값 좌측 조정</div>
그리드 설명
그리드페이지를 참고 해 주세요. 링크
텍스트 컨트롤

1. 정렬

텍스트 정렬은 text-방향을 사용한다. 방향은 총 center, right, left 3가지 이다.

텍스트 중앙
텍스트 오른쪽
텍스트 왼쪽
<div class="text-center">가운데</div>
<div class="text-right">오른쪽</div>
<div class="text-left">왼쪽</div>

2. 컬러

컬러는 text-부트스트랩클래스 를 사용한다.
primary, secondary, success, danger, warning, info, light, dark 자세한사항은 컬러 페이지에서 확인 가능합니다. 링크 

<div class="text-danger">텍스트 danger 컬러</div>
<div class="text-secondary">텍스트 seconday 컬러</div>

3. 텍스트 요소

mark 태그를 사용하여 텍스트를 강조 표시 할 수 있습니다.

삭제된 텍스트로 간주됩니다.

더 이상 정확하지 않은 것으로 간주됩니다.

문서에 추가되는 것으로 간주됩니다.

밑줄이 그어질것입니다

작은 글자로 취급 됩니다.

굵게 표시 됩니다.

기울임 텍스트로 표시 됩니다.

<p>mark 태그를 사용하여 텍스트를 <mark>강조</mark> 표시 할 수 있습니다.</p>
<p><del>삭제된 텍스트로 간주됩니다.</del></p>
<p><s>더 이상 정확하지 않은 것으로 간주됩니다.</s></p>
<p><ins>문서에 추가되는 것으로 간주됩니다.</ins></p>
<p><u>밑줄이 그어질것입니다</u></p>
<p><small>작은 글자로 취급 됩니다.</small></p>
<p><strong>굵게 표시 됩니다.</strong></p>
