사이드 네비게이션 메뉴

사이드 네비게이션 메뉴는 자바스크립트 버전과, 일반 HTML 버전이 있습니다.
URL 엔드포인트에 쿼리스트링이나 다른 분기가 필요한 타입은 일반 HTML을 참고하여 타임리프를 통해 개발을 진행 해 주세요.
그 외에 작업은 자바스크립트 버전으로 작업을 진행 해 주세요.

메뉴 아이콘 변경은 아이콘페이지의 클래스를 참고해서 클래스 명을 변경하여 사용합니다.

자바스크립트 버전 기본 링크 방식으로 사용시
  1. 1. snbControl.js를 참고하여 메뉴를 구성할 수 있습니다.
  2. 2. snbList 변수에 key,value를 수정하여 메뉴를 자동생성 할 수 있습니다.
var snbList = [
    {
        "route": "/index.html", // 라우팅 주소
        "name": "가이드소개", // 메뉴명
        "icon":  "icon-outline-visibility-24px", // 아이콘 클래스네임
    },
    {
        "route": "/pages/guide/learn.html",
        "name": "부트스트랩 시작",
        "icon": "icon-outline-account_circle-24px",
    },
    {
        "route": "#none", // 자식 메뉴가 있을때 #none으로 설정
        "name": "샘플페이지",
        "icon": "icon-outline-assignment-24px",
        "childrenMenu": [
            {
                "route": "/pages/samplePage/page_01.html",
                "name": "기본형",
            },
            {
                "route": "/pages/samplePage/page_01.html",
                "name": "기본형",
                "blank": true, // 새창으로 띄워줄때 값 추가
            }
        ]
    },
]
HTML 버전 타임리프등 템플릿으로 개발시
  1. 1. snbControl.js가 아닌 snbControlHtml.js를 사용합니다.
  2. 2. 하단 코드를 카피하여 인클루드할 파일에 추가하여 작업을 진행한다.
  3. 3. 복사된 HTML코드의 위치는 wrap 클래스 바로 안에 넣어주면 정상 작동 합니다.
