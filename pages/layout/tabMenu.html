탭 메뉴

HTML 버전

block클래스 안에 nav-tabs클래스를 가지고 있는 메뉴를 추가함.
기존의 block-head대신 nav-tabs를 사용한다.

  • 탭 메뉴 2
  • 탭 메뉴 3

<div class="block">
    <ul class="nav-tabs">
        <li class="nav-item active">
            <a href="#">탭 메뉴 1</a>
        </li>
        <li class="nav-item">
            <a href="#">탭 메뉴 2</a>
        </li>
        <li class="nav-item">
            <a href="#">탭 메뉴 3</a>
        </li>
    </ul>
    <div class="block-content">
        내용
    </div>
</div>
