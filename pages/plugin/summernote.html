에디터툴 - summernote

플러그인 사이트 : https://summernote.org

summernote는 부트스트랩을 지원하는 이지윅기반 에디터툴 입니다.

플러그인 추가
기본 CSS와 JS를 추가 해 주세요. (CSS는 head안에 JS는 jQuery하단에 위치하도록 정의) 
// CSS
<link rel="stylesheet" href="https://static.goodchoice.kr/static/ui/css/library/summernote/summernote-bs4.css">

// JS
<script src="https://static.goodchoice.kr/static/ui/js/library/summernote/summernote-bs4.min.js"></script>

추가할 엘리먼트를 지정해 줍니다.

<div id="summernote"></div>

기본 jQuery 선언

$('#summernote').summernote({
    placeholder: '에디터툴 입니다.',
    tabsize: 2,
    height: 300,
});
