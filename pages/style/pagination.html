페이지네이션

설명

페이지 링크는 상황에 따라 사용자 정의가 가능합니다.
li태그에 클래스 disabled와 active로 비활성, 활성 상태를 나타낼 수 있습니다.

페이지네이션의 정렬은 ul태그에 justify-content-center와 justify-content-end를 통해 정렬이 가능합니다.

<ul class="pagination">
    <li class="page-item disabled">
        <a class="page-link" href="#">이전</a>
    </li>
    <li class="page-item">
        <a class="page-link" href="#">1</a>
    </li>
    <li class="page-item active">
        <a class="page-link" href="#">2</a>
    </li>
    <li class="page-item"><a class="page-link" href="#">3</a></li>
    <li class="page-item">
        <a class="page-link" href="#">다음</a>
    </li>
</ul>
