툴팁

기본형

툴팁팝업 사용시 JS호출이 필요합니다.
data-toggle="tooltip"은 고정으로 들어가야하며
data-placement="top" 속성에 top right bottom left 값으로 위치를 지정할 수 있습니다. 

$('[data-toggle="tooltip"]').tooltip();

<button type="button" class="btn btn-secondary" data-toggle="tooltip" data-placement="top" title="툴팁 입니다!">
    툴팁 상단
</button>
<button type="button" class="btn btn-secondary" data-toggle="tooltip" data-placement="right" title="툴팁 입니다!">
    툴팁 우측
</button>
<button type="button" class="btn btn-secondary" data-toggle="tooltip" data-placement="bottom" title="툴팁 입니다!">
    툴팁 하단
</button>
<button type="button" class="btn btn-secondary" data-toggle="tooltip" data-placement="left" title="툴팁 입니다!">
    툴팁 좌측
</button>
커스텀형

해당 태그의 title속성안에 html값을 추가하여 커스텀 툴팁이 제작 가능합니다.
툴팁이 계속 위치하려면 부모 버튼에 클릭이 된 상태여야 노출이 계속 이루어집니다.
데이터 값을 컨트롤해야 하는 작업에는 사용을 지양합니다. 

<button type="button" class="btn btn-danger" data-toggle="tooltip" data-html="true" title='
<a href="/pages/style/grid.html" class="btn btn-sm btn-success">그리드</a>
<a href="/pages/style/color.html" class="btn btn-sm btn-secondary">컬러</a>
<a href="/pages/style/table.html" class="btn btn-sm btn-info">테이블</a>
'>클릭시 노출</button>
드랍다운 형

버튼 클릭시 부모 자식간의 데이터 컨트롤이 필요할 때 사용을 권장합니다.
dropdown-menu-left / dropdown-menu-right로 드랍메뉴 정렬이 가능합니다. 

<button class="btn btn-info" type="button" id="dropdownTest" data-toggle="dropdown" aria-haspopup="true" aria-expanded="true">드랍다운 노출</button>
<div class="dropdown-menu dropdown-menu-left text-center pl-2 pr-2" aria-labelledby="dropdownTest">
    <button type="button" class="btn btn-primary btn-sm">
    <button type="button" class="btn btn-primary btn-sm">
    <button type="button" class="btn btn-primary btn-sm">
</div>
