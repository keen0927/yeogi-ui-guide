모달 팝업

모달은 HTML,CSS,JAVASCRIPT로 작성됩니다.
모달의 "배경"을 클릭하면 자동으로 닫힙니다.
부트스트랩은 한 번에 하나의 모달 창만 지원하며,
중첩 된 모달은 사용자 경험이 좋지 않다고 생각하기 때문에 지원되지 않습니다.

출처 Bootstrap4 공식 메뉴얼 링크

기본설명

data-toggle="modal" 버튼과 같은 컨트롤러 요소에서 data-target="불러올 모달 ID"를 주입하여 불러올 수 있습니다.

// data-target 속성에 불러올 모달ID를 추가
// data-goggle="modal" 속성이 있을 시 별도의 자바스크립트 없이 모달 팝업을 불러올 수 있다.
<button type="button" class="btn btn-primary" data-toggle="modal" data-target="#exampleModal">
    기본형 모달
</button>

<div class="modal fade" id="exampleModal" tabindex="-1">
    <div class="modal-dialog">
        <div class="modal-content">
        <div class="modal-header">
            <h5 class="modal-title" id="exampleModalLabel">모달 제목</h5>
            <button type="button" class="close" data-dismiss="modal">
            <span>×</span>
            </button>
        </div>
        <div class="modal-body">
            내용이 들어갑니다.
        </div>
        <div class="modal-footer">
            <button type="button" class="btn btn-secondary" data-dismiss="modal">닫기</button>
            <button type="button" class="btn btn-primary">저장</button>
        </div>
        </div>
    </div>
</div>

자바스크립트를 통한 호출을 하고 싶은 경우 $('타겟모달').modal('show') / $('타겟모달').modal('hide') 를 통하여 컨트롤이 가능합니다. 

$('타겟모달').modal('show');
$('타겟모달').modal('hide');

또 한 모달의 내용이 현재 디바이스의 뷰포트 높이 보다 길 경우 독립적으로 모달을 스크롤 해 줍니다.

모달의 정렬을 세로의 중앙에 위치하고 싶은 경우 modal-dialog클래스와 동일한 속성에 modal-dialog-centered 클래스 추가로 변경 가능 합니다.

<div class="modal fade" tabindex="-1" id="exampleModalCenter">
    <div class="modal-dialog modal-dialog-centered">
        <div class="modal-content">
        <div class="modal-header">
            <h5 class="modal-title">Modal title</h5>
            <button type="button" class="close" data-dismiss="modal">
                <span>×</span>
            </button>
        </div>
        <div class="modal-body">
            세로 중앙 위치
        </div>
        <div class="modal-footer">
            <button type="button" class="btn btn-secondary" data-dismiss="modal">Close</button>
            <button type="button" class="btn btn-primary">Save changes</button>
        </div>
        </div>
    </div>
</div>
