기본 & 아웃라인 스타일
btn클래스를 기본 선언 후 하단의 클래스들과 조합하여 사용.

btn-primary

btn-secondary

btn-success

btn-info

btn-warning

btn-danger

btn-dark

btn-default

outline스타일은 다음과 같다.

btn-outline-primary

btn-outline-secondary

btn-outline-success

btn-outline-info

btn-outline-warning

btn-outline-danger

btn-outline-dark

// btn클래스는 디폴트로 넣어주고 컬러 및 스타일은 뒤의 클래스를 수정하여 사용.
<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-outline-Success">Success</button>

btn

disabled

// 버튼 스몰
<button class="btn btn-sm btn-primary">Small</button>

// 버튼 노멀
<button class="btn btn-primary">Normal</button>

// 버튼 라지
<button class="btn btn-lg btn-primary">Large</button>

// 버튼 사각형
<button class="btn btn-square btn-primary">Square</button>

// 버튼 라운드형
<button class="btn btn-rounded btn-primary">Rounded</button>

// 버튼 비활성화형
<button class="btn btn-primary" disabled>Square</button>
아이콘스타일

span태그에 아이콘 클래스를 추가한후 button태그 안에 넣어서 사용. 

<button type="button" class="btn btn-success btn-lg">
    <span class="icon-outline-done-24px"></span>
    버튼이름
</button>
