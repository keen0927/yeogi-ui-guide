스피너

스피너는 CSS, JAVASCRIPT로 작성됩니다.
기본설명

1. 자바스크립트를 통한 호출을 하고 싶은 경우 spinner('show') / spinner('hide') 를 통하여 컨트롤이 가능합니다.
2. parameter 값이 없을 경우 ( spinner() ) 'show' 와 동일하게 스피너가 실행됩니다.
3. 스피너를 최상단이나 z-index값을 조절하여 팝업 레이어 위치를 조정하고 싶으면 두번째 파라미터에 z-index값을 추가하여 사용합니다.
4. z-index 값이 높을수록 레이어층의 상단에 위치하게 됨. 기본 1040으로 설정(z-index값은 1만을 넘어가지 않도록 사용합니다) 

spinner();  // 실행   
spinner('show');  // 실행   
spinner('hide');  // 종료   
spinner('show',9999);  // 실행 Z-index값 추가
